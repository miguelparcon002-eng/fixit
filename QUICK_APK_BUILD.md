# ⚡ QUICK APK BUILD - 2 Minutes!

## 🎯 Fastest Way (No Setup Required)

### Just run this ONE command:
```bash
flutter build apk --debug
```

### Where is my APK?
```
build/app/outputs/flutter-apk/app-debug.apk
```

### Share it!
1. Find the file in the folder above
2. Send via WhatsApp, Email, Drive, etc.
3. Done! ✅

---

## 📝 Notes

- File size: ~50-80 MB (larger than release)
- Works perfectly for testing and sharing
- No keystore/signing needed
- Recipients need to enable "Install from Unknown Sources"
